Introduction

Hello, and welcome to the first threejs tutorial... Well, let the learning begins!!!

What is ThreeJS?

ThreeJS is a lightweight JavaScript library used to create 3d content on the web. Creating 3d website would've been daunting if a developer did not make use of a library. To create 3d content on the web , one need to learn alot (3d web, how to use webgpu, e.t.c).

ThreeJs is a library that provides us API's that makes creating 3d content alot easier. (It's also going to help if you know some 3d maths, webgl / webgpu).

So, in short: ThreeJs is a library used to build 3d websites.

Short intro to our project

The project we're going to be building in this tutorial is a basic project that makes use of almost everything you're gonna need to build a 3d website.

let's get started...

Scene, Camera, Renderer

In ThreeJS world... We need a scene, a camera, and a renderer to get anything 3d out of ThreeJs (or anything else). But what is a scene? What is a camera? A renderer?

A scene in ThreeJS is a basic view of where your object stays... It's like a container where you put all your 3d object (so without a scene, there's no place to put our 3d object).

A camera is what we use to picture our scene. We can only view our 3d world if there's a camera... To simplify it, the camera is like the user's eye. What the camera pictures is the only thing our user can see. ThreeJS provides two types of camera. An orthogrgaphic camera (for 2d and 2.5d stuffs) and Perspective camera (for 3d stuffs).

And finally, a renderer renders both the scene (where our object stays) and the camera (our view) to an HTML5 Canvas Element. ThreeJs provides a default canvas's element, or we can manually specify the canvas to be used. (This will make sense as we progress).

Let's get started

Whoopie! First of all, congrats for making it this far :D. I assume by now , we've opened our favourite text editor.

  1. Create a new folder, name it anything you'll remember quick. (e.g My First Threejs Project).
  2. Create a new file in your preferred folder... Name it index with the .html file extension. So the file name(with extension) will be "index.html".
  3. Create a sub-folder named "lib". All our library folders will be here. Download and unzipped the ThreeJs zipped file. (I'm using version r150 for this project...).
  4. Checked the unzipped file, search for three.module.js or three.module.min.js. Alternatively, navigate through : threejs-@version >> build >> three.module.js .
  5. You might notice we're using the module file, instead of other files in the build folder. That's because as of version r150 or so, the ".js" file is depreciated and greatly not recommend... If you're wondering what files are those in the "build folder" that's threejs main file. (we'll need to use one of these files for our project).
  6. Copy the "three.module.js" file to the "lib" folder of our main project. (Check step 1).
  7. Copy the following code to the index.html file:
    <!DOCTYPE html>
    <html lang="en">
        <head>
            <title> ThreeJS Project </title>
        </head>
        <body>
            <script src="js/app.js" type="module" ></script>
        </body>
    </html>
  8. Create another sub-folder in our main project folder. Name it "js"... In the new js folder, create a new file, name it "app.js" (This is where our main program code will go). Type in the following code to the app.js file:
    import * as THREE from "lib/three.module.js";
    console.log(THREE);
  9. Here, we are using Es6 module (note the type="module" in our script tag in index.html)... If you don't know much about it, you can learn it here and come back once you've grasp it. Run the index.html with any web server that you love using... (some editors come with web servers. e.g live server for vs code).
  10. Navigate to the console tab manually or press "f12" (On chrome, I love using it and i greatlt recommend it).
