Introduction

Hello world! Welcome to the official website of Lyricsz Games Store. Not really the homepage... Perhaps, you may love to Check it out :D. Twelve weeks with ThreeJs journey... What was that all about?

The ThreeJs Journey with PJ Dev is a 12 week journey... Where by PJ DEV is expected to learn the "ThreeJs Library" from scratch... Every week he is to create and release a full fledge projects (created using ThreeJS). Well, if that sounds fun to you, learn more in our About section

The Journey

Whoo! Whoo!! Whoo!!! A week A project. How ambitious!!!

Hmm! So, ThreeJS is used for building 3d websites? Right? It's a library mainly used for that right? (More on this later).

Briefly, the journey is to help us all out in choosing our next library for building 3d websites. Every working days per week, PJ Dev is to learn ThreeJS atleast 6hours per day... And every week, let's see what can be built using what he's been taught!!!

Why should i care about all these? Perhaps the best reason is "It let's you figure out whether ThreeJs does has the potential to be amazing for building 3d website or doesn't".

About

I bet we all've been looking forward for this section! The about section... :D

Alternatively to using ThreeJS to build 3d stuffs, one might use Babylon, Playcanvas, e.t.c but all these are libraries... that uses either 3d CSS, WebGPU, WebGL, WebGL2 as their underlying renderer.

Using/Learning pure 3d CSS, WebGL, WebGPU might be as daunting, difficult, cryptic... However we make use of libraries to make stuffs easier for us. No one wants to learn 3d Maths!!!

Libraries are code we make use of. (Other people's code) to further make our coding stuffs easier. They provide sets of API's, Functions e.t.c that does exactly what we're beeing told they would... So, shouldn't everybody be using libraries? Well, it does have it disadvantages... Alot of disadvantages! (just keep that in mind)

But there's BabylonJs, AFrame, Playcanvas... Why ThreeJs? Those are also 3d libraries! Well, you made a point, it's just BabylonJs, Playcanvas (list goes on) was never meant for creating 3d websites... That's where ThreeJS plays the role!

Pros and Cons

This section explains all about the Advantages of following up and Disadvantages of following up.

Advantages
  1. ThreeJS is best for building 3d website.
  2. This website will provide tutorials on how to create almost all the project our dear PJ DEV is going to create... So stay tuned :]
  3. ThreeJS is very lightweight... and has fast renderers.
  4. In terms of web developers favourite library for building 3d stuffs, ThreeJS shine! :D
Disadvantages
  1. ThreeJS lacks Physics support, so stop thinking about building games using "just ThreeJS" (alternative approach include construct 2, Babylon e.t.c.
  2. Inconsistency... ThreeJS is being updated too frequently and this could cause some problems (bugs get attracted).
  3. ThreeJS relies on third party libraries for Physics support (or your library)... and sometimes A won't be talking with B. Since A !== B.
